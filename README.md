# Web Daily Journal

This is a very easy project for beginners.

You can use this repository to:
- Write daily journal notes
- Track learning progress
- Practice editing files
- Build a habit of GitHub commits

How to use:
1. Open any file
2. Add a small journal entry
3. Save the file
4. Commit your change

Goal:
Write something small every day.

Notes:
Write your daily journal entries below.
